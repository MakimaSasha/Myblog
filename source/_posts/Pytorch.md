### 环境搭建

```python
conda create -n pytorch python=3.11	#创建pytorch使用的python环境，版本为3.11
conda activate pytorch	#进入pytorch虚拟环境
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu117	#安装pytorch
conda install nb_conda_kernels		#安装jupyter
jupyter notebook	#打开jupyter
```
