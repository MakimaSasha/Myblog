## 基本操作

``` markdown
终端中使用	vim 文件名	打开文件并进入普通模式，可浏览文件内容，复制，粘贴文件内容
	在普通模式中输入 i 进入编辑模式	输入 : + 命令 可执行命令
常见命令：
	:q	直接退出，一般在没有对文件进行操作时使用
	:wq	保存后退出
	:q!	不保存，直接退出
```

### 普通模式

```markdown
移动光标：
	h	向左移动
	j	向下移动
	k	向上移动
	l	向右移动
	搭配数字使用实现跨跳
	如：先按一下 5 在按一下 j 就是向下跳5行
进阶移动：	
	w		即word跳跃到下一个单词的开头
	b		即begin跳跃到上一个单词的开头
	gg		跳转到文件的第一行
	shift+g	跳转到文件的最后一行
	ctrl+u	向上翻页
	ctrl+d	向下翻页
其他命令：
	f	即find，	查找离光标最近的匹配项	
		使用	f+r	跳转到离光标最近的r上
	y	即yank，	复制。
		使用	yaw	即yank all words 复制整个单词
		使用	yy	复制当前行
		使用	y4j	复制当前行后向下4行的内容
	d	即delete，删除，同y，也可搭配其他
	c	即change，改变
		使用	caw 直接删除光标所在的那个单词，并进入输入模式
		使用	cc	直接删除当前行，并进入输入模式
		使用	c1j 直接删除当前行和向下1行，并进入输入模式
		使用	c4j 直接删除当前行和向下4行，并进入输入模式
	p	即paste，	粘贴
	u	即undo，	撤销
		
```

### 输入模式

```markdown 
在普通模式中
	按下 i 从当前光标位置进入输入模式
	按下 a 从当前光标的后一个位置进入输入模式
	shift+i 从当前光标所在行的开头进入输入模式
	shift+a 从当前光标所在行的末尾进入输入模式
在输入模式中按 esc 退出输入模式，进入普通模式
```

### 命令模式

```markdown
在普通模式下输入	:命令	如：
	:q	直接退出
	:w	保存文件
	:wq	保存并退出
	:PlugInstall	安装插件
	:插件名	打开插件
	?要搜索的内容
```

### 可视模式

```markdown
在普通模式下按	v	进入可视模式	
	在可视模式中移动光标就可以选择内容，选中内容后
		按y复制这段内容
		按d删除这段内容
		按c删除这段内容，并进入输入模式
```

### 其他

```markdown
在普通模式下使用	
	:version	查看vim的配置，包括配置文件的位置。
	:echo $环境变量名	来查看对应环境变量位置
注：
	Windows中的用户的_vimrc文件在用户的家目录下，若没有，则手动创建一个即可。
	若使用插件则可以使用vim-plug来管理插件，需在_vimrc文件中添加
		call plug#begin("存放插件的路径，用户自己指定即可")
		Plug	'插件名'	在vimawesome网站中可以查看
		call plug#end()
	_vimrc文件也可更改快捷键	如：
		imap jk <esc>	用j+k组合键来代替ESC键的功能
		map <silent> <C-e> :NERDTreeToggle<CR>	使用ctrl+e来打开和关闭NERDTreeToggle插件
```
